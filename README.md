# Sales Forecasting for Businesses

## Objective

Build a machine learning model to forecast future sales using historical sales data.

## Dataset

Sample Superstore Dataset

## Tools Used

* Python
* Pandas
* NumPy
* Matplotlib
* Scikit-Learn

## Steps Performed

1. Data Cleaning
2. Feature Engineering
3. Exploratory Data Analysis
4. Train-Test Split
5. Linear Regression Model
6. Model Evaluation
7. Future Sales Forecasting

## Evaluation Metrics

* MAE
* RMSE
* R² Score

## Conclusion

A Linear Regression model was trained to forecast sales. Future sales were predicted and visualized using historical data and generated future dates.

## Future Improvements

* Random Forest Regressor
* XGBoost
* Time Series Forecasting (ARIMA/Prophet)
